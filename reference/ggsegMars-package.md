# ggsegMars: MarsAtlas Cortical Parcellation for the 'ggseg' Ecosystem

MarsAtlas cortical parcellation based on macroanatomical information
with 41 regions per hemisphere for the 'ggseg' ecosystem. Provides a
unified 'ggseg_atlas' object with both 2D polygon geometry and 3D vertex
indices, for use with 'ggseg' and 'ggseg3d'.
